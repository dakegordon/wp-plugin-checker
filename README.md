# WP Plugin Checker

A Chrome extension that checks whether WPForms and WP Mail SMTP are installed on any WordPress site, and displays the installed version.

---

## Install

1. Download the latest `wp-plugin-checker.zip` from the [Releases page](https://github.com/dakegordon/wp-plugin-checker/releases)
2. Unzip the file
3. In Chrome, go to `chrome://extensions`
4. Enable **Developer Mode** (toggle in the top-right corner)
5. Click **Load unpacked** and select the unzipped folder

---

## Update

1. Download the latest zip from the [Releases page](https://github.com/dakegordon/wp-plugin-checker/releases)
2. Unzip, replacing the contents of the existing folder
3. In Chrome, go to `chrome://extensions` and click the **reload icon** on WP Plugin Checker
